---
title: "sessions list"
description: "Gespeicherte MineClawd-Sessions für den aktuellen Owner auflisten."
---

## Zweck

Zeige deine gespeicherten Sessions, damit du sie fortsetzen, löschen oder reparieren kannst.

## Syntax

```bash
/mineclawd sessions
/mineclawd sessions list
```

## Was du bekommst

Die Liste enthält:

- Session-ID
- Session-Titel
- Session-Token
- Zeitpunkt der letzten Aktualisierung
- Markierung der aktiven Session

## Beispiele

```bash
/mineclawd sessions list
```

## Verhalten und Grenzen

- Sessions sind nach Owner-Key getrennt.
- Ausführungen durch Spieler nutzen die Spieler-UUID als Owner-Key.
- Ausführungen über die Konsole nutzen `server` als Owner-Key.

## Häufige Fehler

- Wenn du keine Sessions siehst, erstelle eine mit `/mineclawd sessions new`.
