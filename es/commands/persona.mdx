## Propósito

Usa este comando para revisar o cambiar tu personalidad soul activa.

## Sintaxis

```bash
/mineclawd persona
/mineclawd persona <soul>
```

## Qué hace

- Sin argumentos: muestra el soul activo y los nombres de souls disponibles.
- Con `<soul>`: cambia el soul activo para tu clave de propietario.

## Ejemplos

```bash
/mineclawd persona
/mineclawd persona yuki
/mineclawd persona builder
```

## Comportamiento y límites

- Los nombres de soul se leen desde `mineclawd/souls/*.md`.
- La selección activa se guarda por propietario en `mineclawd/souls/.active/`.
- El cambio de personalidad está bloqueado mientras se ejecuta una solicitud.

## Errores comunes

- `persona not found`: verifica el nombre con `/mineclawd persona`.
- `cannot switch persona while a request is running`: espera a que termine.

Para crear souls, consulta [/configuration/souls](/configuration/souls).
