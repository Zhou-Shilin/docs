MineClawd は、自然な言葉で変更を依頼できる Minecraft Mod です。やりたいことを入力すると、MineClawd があなたのワールドで作業します。

MineClawd には、リアルタイムのストリーミング、Session 管理、assets 追跡を備えた専用オーバーレイ GUI が含まれています。

<Warning>
MineClawd は実験的です。ゲーム内で生成された JavaScript を実行できます。OP 権限と API キーは機密情報として扱ってください。
</Warning>

## 今すぐできること

| これを入力 | 起こること |
| --- | --- |
| `/mclawd summon a villager that trades wooden swords for 64 emerald blocks` | MineClawd が、木の剣を 64 個のエメラルドブロックで交換する村人 NPC を作成します。 |
| `/mclawd tell the whole server in a fancy way that i'm going to sleep` | MineClawd が、あなたが寝ることをサーバー全体へおしゃれな文章で通知します。 |
| `/mclawd always welcomes new players with messages in chat` | MineClawd が、新規プレイヤー参加時に毎回チャットで歓迎メッセージを送ります。 |
| `/mclawd summon god of lightning at random locations every 5 minutes` | MineClawd が、5 分ごとにワールド内のランダム地点へ強力な雷エンティティを召喚します。 |
| `/mclawd make white wool with r=10 continuously spawn under my feet` | MineClawd が、あなたの足元を中心に半径 10 で白い羊毛ブロックを継続生成します。 |
| `/mclawd build a small house at 100, 64, 100` | MineClawd が、指定座標 (100, 64, 100) に小さな家を建築します。 |
