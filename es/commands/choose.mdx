---

## Propósito

Responde a las solicitudes pendientes `ask-user-question` de MineClawd.

## Sintaxis

```bash
/mineclawd choose
/mineclawd choose <option>
```

`<option>` puede ser:

- `1`, `2`, `3`, ... (índice de opción predefinida)
- `other` (ingresar texto personalizado en el chat)
- `skip` (continuar sin responder)
- `cancel` (cancelar la entrada pendiente de texto personalizado)

## Flujo de ejemplo

```bash
/mineclawd choose 2
/mineclawd choose other
# escribe tu respuesta personalizada en el chat normal dentro de 60 segundos
```

## Comportamiento y límites

- Este comando debe ejecutarlo un jugador.
- Solo puedes responder tu propia pregunta pendiente.
- El modo de entrada de texto personalizado vence después de 60 segundos.

## Errores comunes

- `no pending question`: MineClawd no está esperando tu respuesta en este momento.
- `invalid option`: usa autocompletado con tabulación o un número válido.
- `not waiting for custom text input`: ejecutaste `cancel` fuera del modo personalizado.
