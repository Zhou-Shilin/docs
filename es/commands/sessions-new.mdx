---
title: "sessions new"
description: "Crea y cambia a una sesión nueva de MineClawd."
---

## Propósito

Crea una sesión nueva para que las tareas nuevas no se mezclen con contexto anterior.

## Sintaxis

```bash
/mineclawd sessions new
```

## Cuándo usarlo

Úsalo antes de un proyecto nuevo, un estilo de juego distinto o un objetivo diferente.

## Ejemplos

```bash
/mineclawd sessions new
```

## Comportamiento y límites

- La nueva sesión se vuelve tu sesión activa de inmediato.
- El título inicial es `New Session` y puede renombrarse automáticamente después de la primera respuesta.
- El id de sesión es un id corto de 4 caracteres, y el formato del token es `<id>-<title-slug>`.

## Errores comunes

- `cannot create a new session while a request is running`: espera a que termine la tarea actual.
- `failed to create session`: comprueba el acceso de escritura a `mineclawd/sessions/`.
