## Propósito

Usa este comando para pedirle a MineClawd que haga trabajo por ti.

## Sintaxis

```bash
/mineclawd prompt <request>
/mclawd <request>
```

## Cuándo usarlo

Úsalo para solicitudes cotidianas normales, como ediciones del mundo, tareas de comandos o automatización con KubeJS.

## Ejemplos

```bash
/mclawd construye una granja de trigo 9x9 junto a mí
/mineclawd prompt crea un script de servidor que anuncie cuando un jugador se duerme
```

## Comportamiento y límites

- MineClawd valida tu configuración de proveedor antes de comenzar.
- MineClawd crea o reutiliza tu sesión activa automáticamente.
- MineClawd permite una sola solicitud activa por propietario de sesión al mismo tiempo.
- El eco del prompt y la visibilidad del estado de la tarea siguen tu ajuste `broadcast-requests-to`.

## Errores comunes

- `OpenAI API Key is missing` o `Vertex AI API Key is missing`: configura las credenciales en `config`.
- `another request is still running for this session`: espera a que termine y vuelve a intentarlo.
