## ステップ 1: Minecraft 1.20.1 向け Fabric をインストール

1. Fabric Installer をダウンロード: [fabricmc.net/use/installer](https://fabricmc.net/use/installer/)
2. インストーラーを実行します。
3. 次を選びます:
   - Install type: **Client**
   - Minecraft version: **1.20.1**
4. **Install** をクリックします。
5. Minecraft Launcher で **Fabric Loader 1.20.1** を 1 回起動し、ゲームを閉じます。

<Note>
公式ランチャーを使う Windows では、通常はワンクリックでインストールできます。
</Note>

## ステップ 2: 必須 Mod をダウンロード（Fabric + 1.20.1 版）

Modrinth から次のファイルをダウンロードします。各ページで、以下に一致するファイルを選んでください:

- Minecraft version: `1.20.1`
- Loader: `Fabric`

| Mod | ダウンロードリンク |
| --- | --- |
| MineClawd | [modrinth.com/mod/mineclawd](https://modrinth.com/mod/mineclawd) |
| Fabric API | [modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api) |
| KubeJS | [modrinth.com/mod/kubejs](https://modrinth.com/mod/kubejs) |
| Architectury API | [modrinth.com/mod/architectury-api](https://modrinth.com/mod/architectury-api) |
| Rhino | [modrinth.com/mod/rhino](https://modrinth.com/mod/rhino) |
| YetAnotherConfigLib v3 (YACL) | [modrinth.com/mod/yacl](https://modrinth.com/mod/yacl) |
| Mod Menu (GUI 設定用) | [modrinth.com/mod/modmenu](https://modrinth.com/mod/modmenu) |

## ステップ 3: `mods` フォルダーに Mod ファイルを配置

1. Minecraft フォルダーを開きます。
2. `mods` フォルダーを開きます。なければ作成します。
3. ダウンロードした `.jar` ファイルをすべて `mods` に移動します。
4. `.jar` ファイルは解凍しないでください。

## ステップ 4: ゲームを起動して MineClawd GUI を開く

1. **Fabric Loader 1.20.1** プロファイルで Minecraft を起動します。
2. シングルプレイのワールドを作成または開きます。
3. そのワールドで **Allow Cheats** を ON にします。
4. **Mod Menu** -> **MineClawd** -> **Configure** を開きます。

## ステップ 5: AI サービスに接続

まだ AI アカウントがない場合は、OpenRouter で無料モデルを選ぶ方法が簡単です。

1. アカウント作成: [openrouter.ai](https://openrouter.ai/)
2. API キー作成: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)
3. MineClawd GUI で次を設定します:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: OpenRouter キーを貼り付け
   - **Model**: [openrouter.ai/models](https://openrouter.ai/models) からモデル ID を 1 つ選択（無料モデルは `:free` を含むことが多いです）
   - **Summarize Model**: ひとまず同じモデル ID を使います
4. 設定を保存します。

<Info>
無料モデルと制限値は時間とともに変わります。1 つのモデルで失敗する場合は、同じ models ページから別のモデル ID を選んでください。
</Info>

## ステップ 6: 最初のメッセージを送信

ワールドでチャットを開き、次を実行します:

```bash
/mclawd こんにちは
```

設定が正しければ、MineClawd がチャットで返信します。
