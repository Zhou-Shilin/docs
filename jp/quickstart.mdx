## プラットフォームを選ぶ

MineClawd は 3 種類のローダーとバージョンの組み合わせに対応しています。あなたの環境に合うタブを選んでください。

<Tabs>
<Tab title="NeoForge 1.21.1 (推奨)">
### Step 1: NeoForge をインストール

1. [neoforged.net](https://neoforged.net/) から NeoForge Installer をダウンロードします。
2. Minecraft バージョン **1.21.1** を選んでインストーラーを実行します。
3. Minecraft Launcher で **NeoForge 1.21.1** プロファイルを 1 回起動し、ゲームを閉じます。

### Step 2: 必須 Mod をダウンロード

Modrinth から次のファイルを取得します。各ページで **NeoForge** と **1.21.1** に一致するファイルを選んでください。

## Step 3: Mod ファイルを `mods` フォルダーに入れる

1. Minecraft フォルダーを開きます。
2. `mods` フォルダーを開きます。なければ作成します。
3. ダウンロードした `.jar` ファイルをすべて `mods` に移動します。
4. `.jar` ファイルは展開しないでください。

## Step 4: ゲームを起動して MineClawd の設定を開く

1. 選んだ Mod ローダーのプロファイルで Minecraft を起動します。
2. シングルプレイのワールドを作成するか開きます。
3. そのワールドで **Allow Cheats** を ON にします。
4. **Better ModList** -> **MineClawd** -> **Configure** を開きます（Fabric の場合は **Mod Menu**）。

## Step 5: AI サービスを接続する

まだ AI アカウントがない場合は、OpenRouter で無料モデルを選ぶ方法が簡単です。

1. アカウントを作成: [openrouter.ai](https://openrouter.ai/)
2. API キーを作成: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)

<Frame caption="OpenRouter の API キー管理ページ">
  <img src="/images/openrouter.png" alt="API キー管理を表示している OpenRouter 設定ページ" />
</Frame>

3. MineClawd の設定画面で次を設定します。
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: OpenRouter キーを貼り付け
   - **Model**: [openrouter.ai/models](https://openrouter.ai/models) からモデル ID を 1 つ選択（無料モデルには `:free` が含まれることがあります）
   - **Summarize Model**: いったん同じモデル ID を設定
4. 設定を保存します。

<Info>
無料モデルや制限は時間とともに変わります。1 つのモデルで失敗したら、同じ models ページから別のモデル ID を選んでください。
</Info>

### Step 5b (任意): Web 検索を有効化

MineClawd は会話中に Tavily を使って Web 検索できます。これは任意ですが、Mod ドキュメントやクラフトレシピなどのオンライン情報を参照したいときに便利です。

1. Tavily アカウントを作成: [tavily.com](https://tavily.com/)
2. Tavily ダッシュボードで API キーをコピーします。

<Frame caption="Tavily の API キー管理ページ">
  <img src="/images/tavily.png" alt="API キー管理インターフェースを表示する Tavily プラットフォーム" />
</Frame>

3. MineClawd の設定画面で **Search** グループ内の **Tavily API Key** フィールドにキーを貼り付けます。次のコマンドでも設定できます。

```bash
/mineclawd config tavily-api-key <your-tavily-key>
```

設定すると、MineClawd に Web を検索して要点付きソース断片を返す `search` ツールが追加されます。キーが未設定の場合、search ツールは無効で、Web 検索を依頼するとその旨を MineClawd が通知します。

## Step 6: 最初のメッセージを送る

MineClawd と会話する方法は 2 つあります。

### Option A: オーバーレイ GUI を使う

画面左側のフローティングオーブ（Assistive Touch）を探してください。オーブをクリックしてオーバーレイを開き、入力バーにメッセージを入力して **Send** をクリックします。

<Frame caption="Assistive Touch オーブを 1 回クリックするとオーバーレイが開きます">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd Assistive Touch フローティングオーブ" />
</Frame>

### Option B: チャットコマンドを使う

チャットを開いて次を実行します。

```bash
/mclawd hello
```

どちらの方法でも動作します。オーバーレイではストリーミング応答と Session 管理を 1 か所で扱えます。詳しくは [GUI の使い方](/tutorials/using-the-gui) を参照してください。
