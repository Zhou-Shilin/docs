---

## Purpose

Use a shorter command to create and switch to a new session.

## Syntax

```bash
/mineclawd new
```

## Equivalent command

```bash
/mineclawd sessions new
```

## Behavior and limits

This command has the same behavior and limits as `sessions new`.

See [/commands/sessions-new](/commands/sessions-new) for full details.
