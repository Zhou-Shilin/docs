---
title: "sessions repair"
description: "Fehlerhafte Vertex-Function-Call-Turns im Session-Verlauf reparieren."
---

## Zweck

Repariere eine Session, wenn Vertex-Function-Call-Turns fehlerhaft werden.

## Syntax

```bash
/mineclawd sessions repair
/mineclawd sessions repair <session>
```

## Wann du ihn verwendest

Nutze das nach Tool-Call-Mismatch-Fehlern in Vertex-basierten Sessions.

## Beispiele

```bash
/mineclawd sessions repair
/mineclawd sessions repair 1a2b
```

## Verhalten und Grenzen

- Ohne `<session>` repariert MineClawd die aktive Session.
- Mit `<session>` repariert MineClawd die referenzierte Session.
- Wenn keine Reparatur nötig ist, meldet MineClawd diesen Zustand und ändert nichts.

## Häufige Fehler

- `session not found`: Referenz mit `/mineclawd sessions list` prüfen.
- `cannot repair sessions while a request is running`: auf Abschluss warten.
