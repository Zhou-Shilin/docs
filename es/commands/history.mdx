---
title: "history"
description: "Abre una vista legible en libro dentro del juego del historial de la sesión actual."
---

## Propósito

Muestra la conversación de tu sesión activa en una interfaz de libro.

## Sintaxis

```bash
/mineclawd history
```

## Cuándo usarlo

Úsalo para revisar el contexto antes de continuar una tarea larga.

## Comportamiento y límites

- Este comando debe ejecutarlo un jugador, no la consola.
- Se requiere soporte de cliente para el paquete directo de la interfaz de libro de historial.
- Si no hay una sesión activa, MineClawd te pedirá crear una primero.
- El historial muy largo se recorta según los límites del libro.

## Errores comunes

- `this command must be executed by a player`: ejecútalo en el chat del jugador.
- `Client mod is required for /mineclawd history book UI`: instala MineClawd en tu cliente.
- `No active session`: ejecuta primero `/mineclawd sessions new`.
