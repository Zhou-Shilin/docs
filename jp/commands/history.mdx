## 目的

アクティブセッションの会話を本 UI で確認します。

## 構文

```bash
/mineclawd history
```

## 使う場面

長い作業を続ける前に、コンテキストを確認したいときに使います。

## 動作と制限

- このコマンドはコンソールではなくプレイヤーとして実行する必要があります。
- 本 UI パケットを直接開くにはクライアント側サポートが必要です。
- アクティブセッションがない場合、MineClawd は先に作成するよう案内します。
- 履歴が長すぎる場合は本 UI の上限に合わせて切り詰められます。

## よくある失敗

- `this command must be executed by a player`: プレイヤーチャットで実行してください。
- `Client mod is required for /mineclawd history book UI`: クライアントに MineClawd をインストールしてください。
- `No active session`: 先に `/mineclawd sessions new` を実行してください。
