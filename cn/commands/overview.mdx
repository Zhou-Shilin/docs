## 全局规则

- 每个 MineClawd 命令都需要 OP 权限等级 2。
- 命令根路径是 `/mineclawd`。
- `prompt` 的快捷别名是 `/mclawd <request>`。
- 会话引用既支持 session id（`1a2b`），也支持 token（`1a2b-new-session`）。

## 命令列表

| 命令 | 用途 | 参考 |
| --- | --- | --- |
| `/mineclawd prompt <request>` | 向 MineClawd 发送请求。 | [/commands/prompt](/commands/prompt) |
| `/mclawd <request>` | `prompt` 的短别名。 | [/commands/prompt](/commands/prompt) |
| `/mineclawd config ...` | 打开、读取或更新设置。 | [/commands/config](/commands/config) |
| `/mineclawd sessions new` | 创建并切换到新会话。 | [/commands/sessions-new](/commands/sessions-new) |
| `/mineclawd sessions list` | 显示所有已保存会话。 | [/commands/sessions-list](/commands/sessions-list) |
| `/mineclawd sessions resume <session>` | 切换当前活动会话。 | [/commands/sessions-resume](/commands/sessions-resume) |
| `/mineclawd sessions remove <session>` | 删除一个会话。 | [/commands/sessions-remove](/commands/sessions-remove) |
| `/mineclawd sessions repair [session]` | 修复 Vertex 历史中的畸形轮次。 | [/commands/sessions-repair](/commands/sessions-repair) |
| `/mineclawd history` | 打开当前会话历史的书本 UI。 | [/commands/history](/commands/history) |
| `/mineclawd retry <token>` | 重试最近一次失败请求。 | [/commands/retry](/commands/retry) |
| `/mineclawd persona [soul]` | 查看或切换当前 soul 人格。 | [/commands/persona](/commands/persona) |
| `/mineclawd choose [option]` | 回答待处理的澄清问题。 | [/commands/choose](/commands/choose) |
| `/mineclawd new` | `sessions new` 的别名。 | [/commands/new](/commands/new) |
