---
title: "sessions remove"
description: "Elimina una sesión guardada por id o token."
---

## Propósito

Elimina sesiones que ya no necesitas.

## Sintaxis

```bash
/mineclawd sessions remove <session>
```

## Ejemplos

```bash
/mineclawd sessions remove 1a2b
/mineclawd sessions remove 1a2b-my-project-title
```

## Comportamiento y límites

- La eliminación es permanente.
- Si eliminas la sesión activa, MineClawd cambia el puntero activo a la sesión restante actualizada más recientemente.
- Si no queda ninguna sesión, no habrá sesión activa.

## Errores comunes

- `session not found`: revisa el id o token con `/mineclawd sessions list`.
- `cannot remove sessions while a request is running`: espera a que termine.
