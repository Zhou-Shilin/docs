## Ziel

## Waehle deine Plattform

MineClawd unterstuetzt drei Loader- und Versionskombinationen. Waehle den Tab, der zu deinem Setup passt.

## Schritt 3: Mod-Dateien in deinen `mods`-Ordner legen

1. Oeffne deinen Minecraft-Ordner.
2. Oeffne den `mods`-Ordner. Wenn er nicht existiert, erstelle ihn.
3. Verschiebe alle heruntergeladenen `.jar`-Dateien in `mods`.
4. Entpacke `.jar`-Dateien nicht.

## Schritt 4: Spiel starten und MineClawd-Konfiguration oeffnen

1. Starte Minecraft mit deinem Mod-Loader-Profil.
2. Erstelle oder oeffne eine Einzelspielerwelt.
3. Schalte **Allow Cheats** fuer diese Welt auf ON.
4. Oeffne **Better ModList** -> **MineClawd** -> **Configure** (oder **Mod Menu** bei Fabric).

## Schritt 5: AI-Service verbinden

Wenn du noch kein AI-Konto hast, nutze OpenRouter und waehle ein kostenloses Modell.

1. Konto erstellen: [openrouter.ai](https://openrouter.ai/)
2. API-Key erstellen: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)

<Frame caption="OpenRouter API-Key-Verwaltung">
  <img src="/images/openrouter.png" alt="OpenRouter Einstellungsseite mit API-Key-Verwaltung" />
</Frame>

3. Setze im MineClawd-Konfigurationsbildschirm:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: deinen OpenRouter-Key einfuegen
   - **Model**: eine Modell-ID von [openrouter.ai/models](https://openrouter.ai/models) waehlen (kostenlose Modelle enthalten oft `:free`)
   - **Summarize Model**: vorerst dieselbe Modell-ID nutzen
4. Speichere die Konfiguration.

<Info>
Kostenlose Modelle und Limits aendern sich mit der Zeit. Wenn ein Modell fehlschlaegt, waehle eine andere Modell-ID von derselben Seite.
</Info>

### Schritt 5b (optional): Websuche aktivieren

MineClawd kann waehrend Unterhaltungen mit Tavily im Web suchen. Das ist optional, aber nuetzlich, wenn MineClawd Mod-Dokumentation, Crafting-Rezepte oder andere Online-Referenzen nachschlagen soll.

1. Erstelle ein Tavily-Konto: [tavily.com](https://tavily.com/)
2. Kopiere deinen API-Key aus dem Tavily-Dashboard.

<Frame caption="Tavily API-Key-Verwaltung">
  <img src="/images/tavily.png" alt="Tavily API-Plattform mit API-Key-Verwaltung" />
</Frame>

3. Fuege im MineClawd-Konfigurationsbildschirm den Key unter **Search** in das Feld **Tavily API Key** ein. Du kannst den Key auch per Befehl setzen:

```bash
/mineclawd config tavily-api-key <your-tavily-key>
```

Wenn konfiguriert, bekommt MineClawd ein `search`-Tool fuer Webabfragen mit kompakten Quellen-Snippets. Wenn kein Key gesetzt ist, ist das Search-Tool deaktiviert und MineClawd sagt dir das bei einer Websuche.

## Schritt 6: Deine erste Nachricht senden

Du hast zwei Wege, mit MineClawd zu sprechen:

### Option A: Overlay-GUI nutzen

Suche den schwebenden Orb (Assistive Touch) auf der linken Bildschirmseite. Klicke den Orb, um das Overlay zu oeffnen, tippe deine Nachricht in die Eingabeleiste und klicke **Send**.

<Frame caption="Der Assistive Touch Orb oeffnet das Overlay mit einem Klick">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd Assistive Touch Orb" />
</Frame>

### Option B: Chat-Befehle nutzen

Oeffne den Chat und fuehre aus:

```bash
/mclawd hello
```

Beide Methoden funktionieren. Das Overlay bietet Streaming-Antworten und Session-Management an einem Ort. Sieh dir [die GUI verwenden](/tutorials/using-the-gui) fuer die komplette Anleitung an.
