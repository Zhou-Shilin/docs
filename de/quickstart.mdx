## Ziel

Folge dieser Seite Schritt für Schritt, und du bekommst eine erfolgreiche Antwort von MineClawd.

## Schritt 1: Fabric für Minecraft 1.20.1 installieren

1. Lade den Fabric Installer herunter: [fabricmc.net/use/installer](https://fabricmc.net/use/installer/)
2. Starte den Installer.
3. Wähle:
   - Install type: **Client**
   - Minecraft version: **1.20.1**
4. Klicke auf **Install**.
5. Öffne den Minecraft Launcher, starte **Fabric Loader 1.20.1** einmal und schließe danach das Spiel.

<Note>
Unter Windows mit dem offiziellen Launcher ist das normalerweise ein One-Click-Installationsablauf.
</Note>

## Schritt 2: erforderliche Mods herunterladen (Fabric + 1.20.1-Versionen)

Lade diese Dateien von Modrinth herunter. Wähle auf jeder Seite eine Datei, die passt zu:

- Minecraft version: `1.20.1`
- Loader: `Fabric`

| Mod | Download-Link |

| MineClawd | [modrinth.com/mod/mineclawd](https://modrinth.com/mod/mineclawd) |
| Fabric API | [modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api) |

## Schritt 3: Mod-Dateien in deinen `mods`-Ordner legen

1. Öffne deinen Minecraft-Ordner.
2. Öffne den `mods`-Ordner. Wenn er nicht existiert, erstelle ihn.
3. Verschiebe alle heruntergeladenen `.jar`-Dateien nach `mods`.
4. Entpacke `.jar`-Dateien nicht.

## Schritt 4: Spiel starten und MineClawd-GUI öffnen

1. Starte Minecraft mit dem Profil **Fabric Loader 1.20.1**.
2. Erstelle oder öffne eine Einzelspielerwelt.
3. Schalte **Allow Cheats** für diese Welt auf ON.
4. Öffne **Mod Menu** -> **MineClawd** -> **Configure**.

## Schritt 5: einen AI-Service verbinden

Wenn du noch kein AI-Konto hast, nutze OpenRouter und wähle ein kostenloses Modell.

1. Konto erstellen: [openrouter.ai](https://openrouter.ai/)
2. API-Key erstellen: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)
3. In der MineClawd-GUI setzen:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: deinen OpenRouter-Key einfügen
   - **Model**: eine Modell-ID aus [openrouter.ai/models](https://openrouter.ai/models) wählen (kostenlose Modelle enthalten oft `:free`)
   - **Summarize Model**: vorerst dieselbe Modell-ID verwenden
4. Konfiguration speichern.

<Info>
Kostenlose Modelle und Limits ändern sich mit der Zeit. Wenn ein Modell fehlschlägt, wähle auf derselben Modellseite eine andere Modell-ID.
</Info>

## Schritt 6: deine erste Nachricht senden

Öffne den Chat in deiner Welt und führe aus:

```bash
/mclawd hello
```

Wenn das Setup korrekt ist, antwortet MineClawd im Chat.
