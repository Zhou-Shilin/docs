MineClawd 是一个 Minecraft 模组，允许你使用自然语言提出修改请求，MineClawd 会在你的世界中完成这些操作。

从 v1.4.0 开始，MineClawd 提供专属覆盖层 GUI，内置实时流式、会话管理和资产跟踪功能。

<Warning>
MineClawd 当前仍处于实验阶段。它会在游戏内执行生成的 JavaScript。请将 OP 权限和 API 密钥视为敏感信息。
</Warning>

## 你现在可以做什么

| 输入此命令 | 如此生效 |
| --- | --- |
| `/mclawd 召唤一个用 64 个绿宝石块交换木剑的村民` | MineClawd 会在世界中创建一个村民 NPC，提供每把木剑需 64 个绿宝石块的交易。 |
| `/mclawd 以华丽的方式告诉全服我要睡觉` | MineClawd 会向服务器所有玩家广播一条华丽消息，说明你准备睡觉。 |
| `/mclawd 始终在新玩家加入时发送欢迎信息` | MineClawd 会在聊天中欢迎每位新玩家。 |
| `/mclawd 每 5 分钟在随机位置召唤雷神` | MineClawd 会每 5 分钟在世界中随机坐标召唤一个强力雷击实体。 |
| `/mclawd 每秒在我脚下半径 10 生成白色羊毛` | MineClawd 会在你的周围半径 10 的范围内持续生成白色羊毛。 |
| `/mclawd 在 100, 64, 100 位置建一座小屋` | MineClawd 会在指定坐标 (100, 64, 100) 建造一座小房子。 |
