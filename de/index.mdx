MineClawd ist ein Minecraft-Mod, mit dem du Aenderungen in normaler Sprache anfordern kannst. Tippe, was du willst, und MineClawd erledigt die Arbeit in deiner Welt.

MineClawd enthaelt eine eigene Overlay-GUI mit Echtzeit-Streaming, Session-Management und Asset-Tracking.

<Frame caption="Sieh zu, wie MineClawd eine Antwort im Overlay streamt">
  <img src="/images/mineclawd-generating.png" alt="MineClawd Overlay mit gestreamter Antwort" />
</Frame>

<Warning>
MineClawd ist experimentell. Es kann generiertes JavaScript im Spiel ausfuehren. Behandle OP-Zugriff und API-Keys als sensible Daten.
</Warning>

## Was du jetzt tun kannst

| Gib das ein | Was passiert |
| --- | --- |
| `/mclawd summon a villager that trades wooden swords for 64 emerald blocks` | MineClawd erstellt einen Dorfbewohner-NPC, der Holzschwerter fuer je 64 Smaragd-Bloecke anbietet. |
| `/mclawd tell the whole server in a fancy way that i'm going to sleep` | MineClawd sendet eine stilvolle Nachricht an alle Spieler auf dem Server, dass du schlafen gehst. |
| `/mclawd always welcomes new players with messages in chat` | MineClawd sendet eine Begruessung im Chat, sobald ein neuer Spieler die Welt betritt. |
| `/mclawd summon god of lightning at random locations every 5 minutes` | MineClawd beschwoert alle 5 Minuten eine starke Blitz-Entitaet an zufaelligen Orten in der Welt. |
| `/mclawd make white wool with r=10 continuously spawn under my feet` | MineClawd erzeugt dauerhaft weisse Wolle in Radius 10 unter deinen Fuessen. |
| `/mclawd build a small house at 100, 64, 100` | MineClawd baut ein kleines Haus an den Koordinaten (100, 64, 100). |
