---
title: "sessions list"
description: "Lista las sesiones de MineClawd guardadas para el propietario actual."
---

## Propósito

Muestra tus sesiones guardadas para que puedas reanudarlas, eliminarlas o repararlas.

## Sintaxis

```bash
/mineclawd sessions
/mineclawd sessions list
```

## Qué obtienes

La lista incluye:

- id de sesión
- título de la sesión
- token de la sesión
- hora de última actualización
- marcador de activa

## Ejemplos

```bash
/mineclawd sessions list
```

## Comportamiento y límites

- Las sesiones se delimitan por clave de propietario.
- Las ejecuciones de jugador usan el UUID del jugador como clave de propietario.
- Las ejecuciones en consola usan `server` como clave de propietario.

## Errores comunes

- Si no ves sesiones, crea una con `/mineclawd sessions new`.
