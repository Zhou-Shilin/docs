---
title: "sessions repair"
description: "修复会话历史中畸形的 Vertex function-call 轮次。"
---

## 用途

当 Vertex function-call 轮次格式异常时，修复该会话。

## 语法

```bash
/mineclawd sessions repair
/mineclawd sessions repair <session>
```

## 何时使用

在使用 Vertex 的会话出现 tool-call 不匹配错误后使用。

## 示例

```bash
/mineclawd sessions repair
/mineclawd sessions repair 1a2b
```

## 行为和限制

- 不带 `<session>` 时，MineClawd 修复当前活动会话。
- 带 `<session>` 时，MineClawd 修复指定会话。
- 如果无需修复，MineClawd 会告知该状态并保持不变。

## 常见失败

- `session not found`：请用 `/mineclawd sessions list` 验证会话引用。
- `cannot repair sessions while a request is running`：请等待当前请求完成。
