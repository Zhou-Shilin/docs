## 目标

按本页步骤操作，你能获得 MineClawd 的一次成功回复。

## 选择平台

MineClawd 支持三种加载器与版本组合。挑选与你的设置匹配的选项卡。

## 第 3 步：将模组文件放入 `mods` 文件夹

1. 打开你的 Minecraft 文件夹。
2. 进入 `mods` 文件夹；如果不存在，就新建一个。
3. 将所有下载的 `.jar` 文件移动到 `mods`。
4. 不要解压 `.jar` 文件。

## 第 4 步：启动游戏并打开 MineClawd 配置

1. 使用你选择的模组加载器配置启动 Minecraft。
2. 创建或打开一个单人世界。
3. 将该世界的 **允许作弊** 设为 ON。
    4. 打开 **Better ModList** -> **MineClawd** -> **Configure**（Fabric 上则打开 **Mod Menu**）。

## 第 5 步：连接 AI 服务

如果你还没有 AI 账号，可以用 OpenRouter 并选一个免费的模型。

1. 创建账号： [openrouter.ai](https://openrouter.ai/)
2. 创建 API 密钥： [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)
3. 在 MineClawd 配置界面设置：
   - **Provider**：`OpenAI`
   - **Endpoint**：`https://openrouter.ai/api/v1`
   - **API Key**：粘贴你的 OpenRouter 密钥
   - **Model**：从 [openrouter.ai/models](https://openrouter.ai/models) 选择一个模型 ID（免费模型往往带有 `:free`）
   - **Summarize Model**：暂时使用相同的模型 ID
4. 保存配置。

<Info>
免费模型和限制会随时间变化。如果某个模型失效，再从同一页挑另一个模型 ID。
</Info>

## 第 6 步：发送你的第一个消息

你可以通过两种方式与 MineClawd 对话：

### 方式 A：使用覆盖层 GUI

在屏幕左侧找到浮动球体（AssistiveTouch）。点击它展开覆盖层，在输入栏输入消息后点击 **Send**。

<Frame caption="AssistiveTouch 球体一键打开覆盖层">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd AssistiveTouch 浮动球体" />
</Frame>

### 方式 B：使用聊天命令

打开聊天并运行：

```bash
/mclawd hello
```

两种方式都适用。覆盖层提供流式响应与会话管理，详见 [使用 GUI](/tutorials/using-the-gui)。
