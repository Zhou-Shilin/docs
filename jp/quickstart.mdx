## ステップ 3: `mods` フォルダーにファイルを配置

1. Minecraft フォルダーを開きます。
2. `mods` フォルダーを開きます。なければ作成します。
3. ダウンロードした `.jar` ファイルをすべて `mods` に移動します。
4. `.jar` ファイルは解凍しないでください。

## ステップ 4: 起動して MineClawd GUI を開く

1. 選んだローダーのプロファイルで Minecraft を起動します。
2. シングルプレイのワールドを作成または開きます。
3. そのワールドで **Allow Cheats** を ON にします。
4. **Mod Menu** -> **MineClawd** -> **Configure** を開きます。

## ステップ 5: AI サービスに接続

AI アカウントがない場合は OpenRouter で無料モデルを使う方法が取り組みやすいです。

1. [openrouter.ai](https://openrouter.ai/) でアカウントを作成します。
2. [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys) で API キーを発行します。
3. MineClawd 設定画面で次を入力します:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: OpenRouter のキーを貼り付け
   - **Model**: [openrouter.ai/models](https://openrouter.ai/models) からモデル ID を 1 つ選択（無料モデルは `:free` を含むことが多いです）
   - **Summarize Model**: モデル ID を同じにしておきます
4. 設定を保存します。

<Info>
無料モデルと制限は変わります。1 つのモデルで失敗したら同じ models ページから別の ID に切り替えてください。
</Info>

## ステップ 6: 最初のメッセージを送る

### オプション A: オーバーレイ GUI

画面の左に浮かぶ Assistive Touch オーブをクリックするとオーバーレイが開きます。入力バーにメッセージを入力し、**Send** を押します。

<Frame caption="Assistive Touch オーブをクリックしてオーバーレイを開く">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd Assistive Touch 浮遊オーブ" />
</Frame>

### オプション B: チャットコマンド

チャットで次のコマンドを実行します:

```bash
/mclawd こんにちは
```

オーバーレイとチャットコマンドは同じように動作します。オーバーレイではストリーミング出力やセッション管理が使えます。
