MineClawd は、自然な言葉で変更を依頼できる Minecraft Mod です。

やりたいことを入力すると、MineClawd があなたのワールドで作業します。

<Warning>
MineClawd は実験的な機能です。ゲーム内で生成された JavaScript を実行できます。OP 権限と API キーは機密情報として扱ってください。
</Warning>

## 今すぐできること

| これを入力 | 起こること |
| --- | --- |
| `/mclawd 木の剣64本をエメラルドブロック64個と交換する村人を召喚して` | 木の剣を 64 個のエメラルドブロックと交換する村人 NPC を MineClawd が作成します。 |
| `/mclawd これから寝ることをサーバー全体におしゃれな言い方で伝えて` | あなたが寝ることを、サーバー全体におしゃれなメッセージで MineClawd が告知します。 |
| `/mclawd 新しいプレイヤーが参加するたびにチャットで歓迎メッセージを送って` | 新しいプレイヤーが参加するたびに、MineClawd がチャットで歓迎メッセージを送ります。 |
| `/mclawd 5分ごとにランダムな場所へ雷の神を召喚して` | 5 分ごとにランダムな場所へ、強力な雷エンティティを MineClawd が召喚します。 |
| `/mclawd 足元の半径10に白い羊毛を継続生成して` | あなたの足元を中心に半径 10 の範囲で、白い羊毛ブロックを MineClawd が継続的に生成します。 |
| `/mclawd 100, 64, 100 に小さな家を建てて` | 指定座標 (100, 64, 100) に小さな家を MineClawd が建築します。 |
