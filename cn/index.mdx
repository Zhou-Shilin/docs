MineClawd 是一个 Minecraft 模组，让你可以用自然语言提出修改请求。

你输入想要的内容，MineClawd 就能在你的世界里执行。

<Warning>
MineClawd 仍处于实验阶段。它可以在游戏内执行生成的 JavaScript。请将 OP 权限和 API Key 视为敏感信息。
</Warning>

## 你现在可以做什么

| 输入这个 | 会发生什么 |
| --- | --- |
| `/mclawd 召唤一个村民，用 64 个绿宝石方块交换木剑` | MineClawd 会创建一个村民 NPC，用 64 个绿宝石方块兑换一把木剑。 |
| `/mclawd 用华丽一点的方式告诉全服我要去睡觉了` | MineClawd 会向全服玩家广播一条更有风格的消息，宣布你要去睡觉了。 |
| `/mclawd 每次有新玩家加入都在聊天栏欢迎他` | 每当有新玩家加入世界时，MineClawd 都会在聊天栏发送欢迎消息。 |
| `/mclawd 每隔 5 分钟在随机位置召唤雷电之神` | MineClawd 会每 5 分钟在 Minecraft 世界随机位置召唤一个强力雷电实体。 |
| `/mclawd 持续在我脚下生成 r=10 的白色羊毛` | MineClawd 会以你为中心半径 10 持续生成白色羊毛方块，位置在你脚下。 |
| `/mclawd 在 100, 64, 100 建一间小房子` | MineClawd 会在指定坐标 (100, 64, 100) 在 Minecraft 世界中建造一间小房子。 |
