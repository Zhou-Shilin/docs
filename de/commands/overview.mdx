## Globale Regeln

- Jeder MineClawd-Befehl benötigt OP-Berechtigungsstufe 2.
- Die Befehlswurzel ist `/mineclawd`.
- Das schnelle Alias für Prompt ist `/mclawd <request>`.
- Session-Referenzen akzeptieren entweder eine Session-ID (`1a2b`) oder ein Token (`1a2b-new-session`).

## Befehlsübersicht

| Befehl | Zweck | Referenz |
| --- | --- | --- |
| `/mineclawd prompt <request>` | Eine Anfrage an MineClawd senden. | [/commands/prompt](/commands/prompt) |
| `/mclawd <request>` | Kurzes Alias für `prompt`. | [/commands/prompt](/commands/prompt) |
| `/mineclawd config ...` | Einstellungen öffnen, lesen oder aktualisieren. | [/commands/config](/commands/config) |
| `/mineclawd sessions new` | Eine neue Session erstellen und zu ihr wechseln. | [/commands/sessions-new](/commands/sessions-new) |
| `/mineclawd sessions list` | Alle gespeicherten Sessions anzeigen. | [/commands/sessions-list](/commands/sessions-list) |
| `/mineclawd sessions resume <session>` | Aktive Session wechseln. | [/commands/sessions-resume](/commands/sessions-resume) |
| `/mineclawd sessions remove <session>` | Eine Session löschen. | [/commands/sessions-remove](/commands/sessions-remove) |
| `/mineclawd sessions repair [session]` | Fehlerhafte Vertex-Verlaufs-Turns reparieren. | [/commands/sessions-repair](/commands/sessions-repair) |
| `/mineclawd history` | Die aktuelle Session-Verlauf-Buch-UI öffnen. | [/commands/history](/commands/history) |
| `/mineclawd retry <token>` | Letzte fehlgeschlagene Anfrage erneut ausführen. | [/commands/retry](/commands/retry) |
| `/mineclawd persona [soul]` | Aktive Soul-Persona anzeigen oder wechseln. | [/commands/persona](/commands/persona) |
| `/mineclawd choose [option]` | Ausstehende Rückfragen beantworten. | [/commands/choose](/commands/choose) |
| `/mineclawd new` | Alias von `sessions new`. | [/commands/new](/commands/new) |
