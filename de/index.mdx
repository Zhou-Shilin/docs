MineClawd ist ein Minecraft-Mod, mit dem du Änderungen in normaler Sprache anfordern kannst.

Du schreibst, was du willst. MineClawd kann die Arbeit in deiner Welt übernehmen.

<Warning>
MineClawd ist experimentell. Es kann generiertes JavaScript im Spiel ausführen. Behandle OP-Zugriff und API-Keys als sensible Daten.
</Warning>

## Was du jetzt schon tun kannst

| Gib das ein | Was passiert |
| --- | --- |
| `/mclawd beschwöre einen Dorfbewohner, der Holzschwerter für 64 Smaragdblöcke handelt` | MineClawd erstellt einen Dorfbewohner-NPC, der pro Angebot Holzschwerter gegen 64 Smaragdblöcke tauscht. |
| `/mclawd sage dem ganzen Server auf elegante Weise, dass ich schlafen gehe` | MineClawd sendet eine stilvolle Nachricht an alle Spieler auf dem Server und kündigt an, dass du schlafen gehst. |
| `/mclawd begrüße neue Spieler immer mit einer Nachricht im Chat` | MineClawd sendet jedes Mal eine Willkommensnachricht im Chat, wenn ein neuer Spieler die Welt betritt. |
| `/mclawd beschwöre alle 5 Minuten einen Blitzgott an zufälligen Orten` | MineClawd beschwört alle 5 Minuten eine mächtige Blitz-Entität an zufälligen Orten in der Minecraft-Welt. |
| `/mclawd lasse unter meinen Füßen fortlaufend weiße Wolle mit r=10 erscheinen` | MineClawd erzeugt weiße Wollblöcke in einem Radius von 10 um dich herum und lässt sie fortlaufend unter deinen Füßen erscheinen. |
| `/mclawd baue ein kleines Haus bei 100, 64, 100` | MineClawd baut ein kleines Haus an den angegebenen Koordinaten (100, 64, 100) in der Minecraft-Welt. |
