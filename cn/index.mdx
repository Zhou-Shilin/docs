MineClawd 是一个 Minecraft 模组，可让你用自然语言提出修改请求。输入你想要的结果，MineClawd 会在你的世界中执行。

MineClawd 内置专用覆盖层 GUI，提供实时流式响应、Session 管理和资产追踪。

  <img src="/images/mineclawd-generating.png" alt="MineClawd 覆盖层 GUI 显示流式响应" />

<Warning>
MineClawd 仍处于实验阶段。它会在游戏内执行生成的 JavaScript。请将 OP 权限和 API 密钥视为敏感信息。
</Warning>

## 你现在可以做什么

| 输入这句话 | 会发生什么 |
| --- | --- |
| `/mclawd 召唤一个用 64 个绿宝石块交换木剑的村民` | MineClawd 会创建一个村民 NPC，提供每把木剑需 64 个绿宝石块的交易。 |
| `/mclawd 以华丽的方式告诉全服我要睡觉` | MineClawd 会向服务器所有玩家广播一条风格化消息，说明你要去睡觉。 |
| `/mclawd 始终在新玩家加入时发送欢迎信息` | MineClawd 会在新玩家加入世界时自动发送欢迎消息。 |
| `/mclawd 每 5 分钟在随机位置召唤雷神` | MineClawd 会每 5 分钟在随机位置召唤强力雷击实体。 |
| `/mclawd 每秒在我脚下半径 10 生成白色羊毛` | MineClawd 会在你脚下周围半径 10 的范围持续生成白色羊毛。 |
| `/mclawd 在 100, 64, 100 位置建一座小屋` | MineClawd 会在坐标 (100, 64, 100) 建造一座小房子。 |
