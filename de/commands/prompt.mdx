---

## Zweck

Nutze diesen Befehl, um MineClawd Arbeit für dich erledigen zu lassen.

## Syntax

```bash
/mineclawd prompt <request>
/mclawd <request>
```

## Wann du ihn verwendest

Nutze das für normale tägliche Anfragen, etwa Weltbearbeitungen, Befehlsaufgaben oder KubeJS-Automatisierung.

## Beispiele

```bash
/mclawd baue neben mir eine 9x9-Weizenfarm
/mineclawd prompt erstelle ein Server-Skript, das meldet, wenn ein Spieler schläft
```

## Verhalten und Grenzen

- MineClawd validiert vor dem Start deine Provider-Konfiguration.
- MineClawd erstellt oder verwendet deine aktive Session automatisch wieder.
- MineClawd erlaubt jeweils nur eine aktive Anfrage pro Session-Owner.
- Prompt-Echo und Sichtbarkeit des Aufgabenstatus folgen deiner Einstellung `broadcast-requests-to`.

## Häufige Fehler

- `OpenAI API Key is missing` oder `Vertex AI API Key is missing`: Anmeldedaten in der Konfiguration setzen.
- `another request is still running for this session`: auf Abschluss warten und dann erneut versuchen.
