## Propósito

Usa un comando más corto para crear y cambiar a una sesión nueva.

## Sintaxis

```bash
/mineclawd new
```

## Comando equivalente

```bash
/mineclawd sessions new
```

## Comportamiento y límites

Este comando tiene el mismo comportamiento y límites que `sessions new`.

Consulta [/commands/sessions-new](/commands/sessions-new) para ver todos los detalles.
