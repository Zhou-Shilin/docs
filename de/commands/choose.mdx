---

## Zweck

Ausstehende `ask-user-question`-Prompts von MineClawd beantworten.

## Syntax

```bash
/mineclawd choose
/mineclawd choose <option>
```

`<option>` kann sein:

- `1`, `2`, `3`, ... (vordefinierter Optionsindex)
- `other` (eigenen Chat-Text eingeben)
- `skip` (ohne Antwort fortfahren)
- `cancel` (ausstehende Eingabe von eigenem Text abbrechen)

## Beispielablauf

```bash
/mineclawd choose 2
/mineclawd choose other
# gib deine eigene Antwort innerhalb von 60 Sekunden im normalen Chat ein
```

## Verhalten und Grenzen

- Dieser Befehl muss von einem Spieler ausgeführt werden.
- Du kannst nur deine eigene ausstehende Frage beantworten.
- Der Modus für die Eingabe von eigenem Text läuft nach 60 Sekunden ab.

## Häufige Fehler

- `no pending question`: MineClawd wartet aktuell nicht auf deine Eingabe.
- `invalid option`: nutze Tab-Vervollständigung oder eine gültige Zahl.
- `not waiting for custom text input`: du hast `cancel` außerhalb des benutzerdefinierten Modus aufgerufen.
