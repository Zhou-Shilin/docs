## 目标

按本页步骤操作后，你会拿到 MineClawd 的一次成功回复。

## 选择平台

MineClawd 支持三种加载器与版本组合。请选择与你环境匹配的标签页。

<Tabs>
<Tab title="NeoForge 1.21.1 (推荐)">
### 第 1 步：安装 NeoForge

1. 从 [neoforged.net](https://neoforged.net/) 下载 NeoForge Installer。
2. 选择 Minecraft 版本 **1.21.1**，然后运行安装器。
3. 打开 Minecraft Launcher，先运行一次 **NeoForge 1.21.1** 配置，再关闭游戏。

## 第 3 步：把模组文件放进 `mods` 文件夹

1. 打开 Minecraft 文件夹。
2. 打开 `mods` 文件夹；如果不存在就创建。
3. 把所有下载的 `.jar` 文件移动到 `mods`。
4. 不要解压 `.jar` 文件。

## 第 4 步：启动游戏并打开 MineClawd 配置

1. 使用你的模组加载器配置启动 Minecraft。
2. 创建或打开一个单人世界。
3. 将该世界的 **Allow Cheats** 设为 ON。
4. 打开 **Better ModList** -> **MineClawd** -> **Configure**（Fabric 请使用 **Mod Menu**）。

## 第 5 步：连接 AI 服务

如果你还没有 AI 账号，建议先用 OpenRouter 并选择免费模型。

1. 创建账号：[openrouter.ai](https://openrouter.ai/)
2. 创建 API key：[openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)

<Frame caption="OpenRouter API key 管理页面">
  <img src="/images/openrouter.png" alt="OpenRouter 设置页面中的 API key 管理区域" />
</Frame>

3. 在 MineClawd 配置页设置：
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: 粘贴你的 OpenRouter key
   - **Model**: 从 [openrouter.ai/models](https://openrouter.ai/models) 选择一个 model id（免费模型常带 `:free`）
   - **Summarize Model**: 先用同一个 model id
4. 保存配置。

<Info>
免费模型和限制会变化。如果某个模型失败，请在同一模型页换一个 model id。
</Info>

### 第 5b 步（可选）：启用网页搜索

MineClawd 可通过 Tavily 在对话中搜索网页。当你需要查询模组文档、合成配方或其他在线资料时，这很有用。

1. 创建 Tavily 账号：[tavily.com](https://tavily.com/)
2. 在 Tavily 控制台复制你的 API key。

<Frame caption="Tavily API key 管理页面">
  <img src="/images/tavily.png" alt="Tavily API 平台中的 API key 管理界面" />
</Frame>

3. 在 MineClawd 配置页 **Search** 分组下，将 key 填入 **Tavily API Key**。你也可以用命令设置：

```bash
/mineclawd config tavily-api-key <your-tavily-key>
```

配置完成后，MineClawd 会获得 `search` 工具来查询网页并返回精简来源片段。如果未设置 key，`search` 工具会禁用，MineClawd 会在你请求网页搜索时明确提示。

## 第 6 步：发送你的第一条消息

你可以通过两种方式与 MineClawd 对话：

### 方式 A：使用覆盖层 GUI

在屏幕左侧找到浮动球（Assistive Touch）。点击球体打开覆盖层，在输入栏输入消息后点击 **Send**。

<Frame caption="Assistive Touch 浮动球一键打开覆盖层">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd Assistive Touch 浮动球" />
</Frame>

### 方式 B：使用聊天命令

打开聊天并运行：

```bash
/mclawd hello
```

两种方式都可用。覆盖层把流式响应和 Session 管理放在同一处。完整用法见 [使用 GUI](/tutorials/using-the-gui)。
