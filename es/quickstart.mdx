## Objetivo

Sigue esta página paso a paso y recibirás una respuesta exitosa de MineClawd.

## Glosario rápido

<CardGroup cols={2}>
  <Card title="Minecraft Java Edition">
    La versión oficial para PC del juego desde el Minecraft Launcher.
  </Card>
  <Card title="Mod loader">
    Software que permite ejecutar mods. MineClawd es compatible con Fabric, Forge y NeoForge.
  </Card>
  <Card title="Mod">
    Un archivo `.jar` que añade funciones a Minecraft.
  </Card>
  <Card title="Clave API">

  </Card>

## Elige tu plataforma

## Paso 3: coloca los archivos del mod en tu carpeta `mods`

1. Abre tu carpeta de Minecraft.
2. Abre la carpeta `mods`. Si no existe, créala.
3. Mueve todos los archivos `.jar` descargados a `mods`.
4. No descomprimas los archivos `.jar`.

## Paso 4: inicia el juego y abre la configuración de MineClawd

1. Inicia Minecraft con el perfil del loader elegido.
2. Crea o abre un mundo de un solo jugador.
3. Activa **Allow Cheats** en ese mundo.
4. Abre **Better ModList** -> **MineClawd** -> **Configure** (o **Mod Menu** si usas Fabric).

## Paso 5: conecta un servicio de IA

Si aún no tienes cuenta, usa OpenRouter y elige un modelo gratuito.

1. Crea una cuenta en [openrouter.ai](https://openrouter.ai/).
2. Crea una clave API en [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys).
3. En la pantalla de configuración de MineClawd, define:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: pega tu clave de OpenRouter
   - **Model**: elige un id de modelo de [openrouter.ai/models](https://openrouter.ai/models) (los modelos gratuitos suelen incluir `:free`)
   - **Summarize Model**: usa por ahora el mismo id de modelo
4. Guarda la configuración.

<Info>
Los modelos gratuitos y sus límites cambian con el tiempo. Si un modelo falla, elige otro id de la misma página.
</Info>

## Paso 6: envía tu primer mensaje

Tienes dos formas de hablar con MineClawd:

### Opción A: usa la GUI de superposición

Busca el orbe flotante (AssistiveTouch) en el lado izquierdo de la pantalla. Haz clic en él, escribe tu mensaje en la barra de entrada y haz clic en **Send**.

<Frame caption="El orbe AssistiveTouch abre la superposición con un clic">
  <img src="/images/mineclawd-assistive-touch.png" alt="El orbe flotante AssistiveTouch de MineClawd" />
</Frame>

### Opción B: usa comandos en chat

Abre el chat y ejecuta:

```bash
/mclawd hello
```

Ambos métodos funcionan. La GUI ofrece respuestas en streaming y gestión de sesiones en un solo lugar. Consulta [Uso de la GUI](/tutorials/using-the-gui) para la guía completa.
