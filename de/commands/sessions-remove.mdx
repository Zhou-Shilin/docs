---
title: "sessions remove"
description: "Eine gespeicherte Session per ID oder Token löschen."
---

## Zweck

Entferne Sessions, die du nicht mehr brauchst.

## Syntax

```bash
/mineclawd sessions remove <session>
```

## Beispiele

```bash
/mineclawd sessions remove 1a2b
/mineclawd sessions remove 1a2b-my-project-title
```

## Verhalten und Grenzen

- Das Löschen ist dauerhaft.
- Wenn du die aktive Session entfernst, setzt MineClawd den aktiven Zeiger auf die zuletzt aktualisierte verbleibende Session.
- Wenn keine Sessions übrig bleiben, gibt es keine aktive Session.

## Häufige Fehler

- `session not found`: ID oder Token mit `/mineclawd sessions list` prüfen.
- `cannot remove sessions while a request is running`: auf Abschluss warten.
