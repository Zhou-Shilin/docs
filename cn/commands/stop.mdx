## 目的

取消会话中正在运行的请求。

## 语法

```bash
/mineclawd stop
```

## 何时使用

- 请求运行时间过长，需要中止。
- MineClawd 卡在工具调用循环。
- 你在生成途中改变主意。

## 执行内容

1. 将当前请求标记为已取消。
2. 终止发送到 LLM 服务的网络请求。
3. 清理任何待处理的 `ask-user-question` 提示。
4. 释放会话，以便发送新请求。

## GUI 替代操作

请求运行时，覆盖层输入栏会将 **Send** 按钮换成带正方形图标的 **Stop** 按钮。点击该按钮与运行 `/mineclawd stop` 效果相同。

## 行为与限制

- 如果没有运行中的请求，命令不会产生效果。
- 取消前已经流式展示的部分响应仍保留在历史中。
- 取消后会话仍保持原活动状态。

## 常见失败

- `No active request to cancel` —— 说明当前会话没有正在生成的请求。
