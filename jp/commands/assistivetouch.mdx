## 目的

オーバーレイ全体を無効化せずに、Assistive Touch オーブの表示を切り替えます。

## 構文

```bash
/mineclawd assistivetouch
```

## 使うタイミング

- オーブを一時的に画面から隠したいとき。
- Overlay を別経路で開く（たとえばセッションやアセットメニュー）ことが多いとき。

## 動作

プレイヤーごとの表示状態をトグルします。一度実行すると非表示、もう一度で再表示します。状態は `mineclawd/player-settings.json` に保存され、セッションを跨いで保持されます。

## 制限

- 状態はプレイヤー単位で、サーバー全体には影響しません。
- オーバーレイがすでに開いている場合は閉じません。
- 表示は OP 権限レベル 2 があるプレイヤーのみに限定されます。
- 初期状態は表示（有効）です。
