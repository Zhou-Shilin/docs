---
title: "sessions repair"
description: "Repara turnos mal formados de llamadas de función de Vertex en el historial de sesión."
---

## Propósito

Repara una sesión cuando los turnos de llamadas de función de Vertex quedan mal formados.

## Sintaxis

```bash
/mineclawd sessions repair
/mineclawd sessions repair <session>
```

## Cuándo usarlo

Úsalo después de errores de desajuste de llamadas de herramientas en sesiones respaldadas por Vertex.

## Ejemplos

```bash
/mineclawd sessions repair
/mineclawd sessions repair 1a2b
```

## Comportamiento y límites

- Sin `<session>`, MineClawd repara la sesión activa.
- Con `<session>`, MineClawd repara la sesión referenciada.
- Si no se necesita reparación, MineClawd informa ese estado y no cambia nada.

## Errores comunes

- `session not found`: verifica la referencia con `/mineclawd sessions list`.
- `cannot repair sessions while a request is running`: espera a que termine.
