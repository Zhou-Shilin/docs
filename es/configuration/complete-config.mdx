## Propósito

Esta página te da el modelo completo de configuración de MineClawd, incluyendo ajustes de archivo, claves de comando y comportamiento de reinicio.

## Dónde están los ajustes

- Archivo de configuración global: `config/mineclawd.json5`
- Visibilidad del chat por jugador: `mineclawd/player-settings.json`
- Archivos soul y estado del soul activo: `mineclawd/souls/` y `mineclawd/souls/.active/`

## Plantilla completa de `mineclawd.json5`

```json5
{
  provider: "OPENAI",
  endpoint: "https://api.openai.com/v1",
  apiKey: "",
  model: "gpt-4o-mini",
  summarizeModel: "gpt-4o-mini",

  vertexEndpoint: "https://aiplatform.googleapis.com/v1",
  vertexApiKey: "",
  vertexModel: "publishers/google/models/gemini-3-pro-preview",
  vertexSummarizeModel: "publishers/google/models/gemini-3-pro-preview",

  debugMode: false,
  limitToolCalls: false,
  toolCallLimit: 16,
  systemPrompt: "",
  dynamicRegistryMode: "AUTO"
}
```

<Info>
En formato de archivo, los valores enum van en mayúsculas (`OPENAI`, `VERTEX_AI`, `AUTO`, `ENABLED`, `DISABLED`).

En formato de comando, los valores van en minúsculas (`openai`, `vertex-ai`, `auto`, `enabled`, `disabled`).
</Info>

## Referencia de claves

| Clave de archivo | Clave de comando | Tipo | Predeterminado | Notas |
| --- | --- | --- | --- | --- |
| `provider` | `provider` | enum | `OPENAI` | Selección de proveedor: compatible con OpenAI o Vertex AI. |
| `endpoint` | `endpoint` | string | `https://api.openai.com/v1` | URL base compatible con OpenAI; no requiere barra final. |
| `apiKey` | `api-key` | string | vacío | Clave API de OpenAI. |
| `model` | `model` | string | `gpt-4o-mini` | Modelo de chat de OpenAI. |
| `summarizeModel` | `summarize-model` | string | `gpt-4o-mini` | Modelo de OpenAI para generar el título de sesión en el primer turno. |
| `vertexEndpoint` | `vertex-endpoint` | string | `https://aiplatform.googleapis.com/v1` | URL base de la API de Vertex. |
| `vertexApiKey` | `vertex-api-key` | string | vacío | Clave API de Vertex. |
| `vertexModel` | `vertex-model` | string | `publishers/google/models/gemini-3-pro-preview` | Ruta del modelo de Vertex o nombre de modelo. |
| `vertexSummarizeModel` | `vertex-summarize-model` | string | `publishers/google/models/gemini-3-pro-preview` | Modelo de resumen de Vertex para títulos de sesión. |
| `debugMode` | `debug-mode` | boolean | `false` | Registra actividad detallada de solicitudes y herramientas. |
| `limitToolCalls` | `limit-tool-calls` | boolean | `false` | Habilita un límite de bucle de herramientas por solicitud. |
| `toolCallLimit` | `tool-call-limit` | integer | `16` | El rango permitido es de `1` a `20`. |
| `systemPrompt` | `system-prompt` | string | vacío | Vacío significa usar el system prompt incorporado. Usa `default` en modo comando para restablecer. |
| `dynamicRegistryMode` | `dynamic-registry-mode` | enum | `AUTO` | Controla el modo de ejecución de marcadores dinámicos. Requiere reinicio después del cambio. |

## Ajuste por jugador

`broadcast-requests-to` es solo de comando y por jugador.

- Clave de comando: `broadcast-requests-to`
- Valores permitidos: `self`, `all`, `ops`
- Se guarda en: `mineclawd/player-settings.json`
- Propósito: controla quién ve el eco del prompt y las líneas de inicio o fin de tarea.

## Comportamiento del modo de registro dinámico

- `AUTO`: habilitado en runtime de cliente (un jugador), deshabilitado en servidores dedicados.
- `ENABLED`: forzado como habilitado en todos los runtimes.
- `DISABLED`: totalmente deshabilitado.

<Warning>
Si fuerzas `ENABLED` en un servidor dedicado, los clientes que se conecten deben tener MineClawd instalado.
</Warning>

## Configuraciones base recomendadas

### Base para una primera configuración con OpenAI

```bash
/mineclawd config provider openai
/mineclawd config endpoint https://api.openai.com/v1
/mineclawd config api-key <your-key>
/mineclawd config model gpt-4o-mini
/mineclawd config summarize-model gpt-4o-mini
```

### Base para una primera configuración con Vertex

```bash
/mineclawd config provider vertex-ai
/mineclawd config vertex-endpoint https://aiplatform.googleapis.com/v1
/mineclawd config vertex-api-key <your-key>
/mineclawd config vertex-model publishers/google/models/gemini-3-pro-preview
/mineclawd config vertex-summarize-model publishers/google/models/gemini-3-pro-preview
```

## Ruta GUI

- En el juego: **Mod Menu** -> **MineClawd** -> **Configure**
- Alternativa por comando: `/mineclawd config ...`

Si falla la apertura de la GUI en cliente, configura mediante claves de comando.
