## Objetivo

Sigue esta página paso a paso y obtendrás una respuesta exitosa de MineClawd.

## Glosario rápido

<CardGroup cols={2}>
  <Card title="Minecraft Java Edition">
    La versión para PC de Minecraft desde el launcher oficial.
  </Card>
  <Card title="Mod loader">
    Software que permite que Minecraft ejecute mods. MineClawd soporta Fabric, Forge y NeoForge.
  </Card>
  <Card title="Mod">
    Un archivo `.jar` que añade funciones a Minecraft.
  </Card>
  <Card title="API key">
    Una clave secreta que permite que MineClawd se conecte a un servicio de IA en línea.
  </Card>
</CardGroup>

## Elige tu plataforma

MineClawd soporta tres combinaciones de loader y versión. Elige la pestaña que coincida con tu setup.

<Tabs>
<Tab title="NeoForge 1.21.1 (Recomendado)">
### Paso 1: instala NeoForge

1. Descarga NeoForge Installer desde [neoforged.net](https://neoforged.net/)
2. Selecciona la versión de Minecraft **1.21.1** y ejecuta el instalador.
3. Abre Minecraft Launcher y ejecuta el perfil **NeoForge 1.21.1** una vez, luego cierra el juego.

## Paso 3: coloca los archivos del mod en tu carpeta `mods`

1. Abre tu carpeta de Minecraft.
2. Abre la carpeta `mods`. Si no existe, créala.
3. Mueve todos los archivos `.jar` descargados a `mods`.
4. No descomprimas los archivos `.jar`.

## Paso 4: inicia el juego y abre la configuración de MineClawd

1. Inicia Minecraft con tu perfil de mod loader.
2. Crea o abre un mundo de un jugador.
3. Activa **Allow Cheats** en ese mundo.
4. Abre **Better ModList** -> **MineClawd** -> **Configure** (o **Mod Menu** si usas Fabric).

## Paso 5: conecta un servicio de IA

Si todavía no tienes una cuenta de IA, usa OpenRouter y elige un modelo gratuito.

1. Crea una cuenta: [openrouter.ai](https://openrouter.ai/)
2. Crea una API key: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)

<Frame caption="Página de gestión de API keys de OpenRouter">
  <img src="/images/openrouter.png" alt="Página de configuración de OpenRouter mostrando gestión de API keys" />
</Frame>

3. En la pantalla de configuración de MineClawd, define:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: pega tu clave de OpenRouter
   - **Model**: elige un model id de [openrouter.ai/models](https://openrouter.ai/models) (los modelos gratis suelen incluir `:free`)
   - **Summarize Model**: usa el mismo model id por ahora
4. Guarda la configuración.

<Info>
Los modelos gratis y sus límites cambian con el tiempo. Si un modelo falla, elige otro model id en la misma página.
</Info>

### Paso 5b (opcional): habilita búsqueda web

MineClawd puede buscar en la web durante las conversaciones con Tavily. Es opcional, pero útil cuando quieres que MineClawd consulte documentación de mods, recetas de crafteo u otras referencias online.

1. Crea una cuenta en Tavily: [tavily.com](https://tavily.com/)
2. Copia tu API key desde el panel de Tavily.

<Frame caption="Página de gestión de API keys de Tavily">
  <img src="/images/tavily.png" alt="Plataforma de Tavily mostrando la interfaz de gestión de API keys" />
</Frame>

3. En la pantalla de configuración de MineClawd, pega la clave en **Tavily API Key** dentro del grupo **Search**. También puedes configurarla con este comando:

```bash
/mineclawd config tavily-api-key <your-tavily-key>
```

Cuando se configura, MineClawd obtiene la herramienta `search` para consultar la web y devolver fragmentos concisos con fuentes. Si la clave no está definida, la herramienta de búsqueda se desactiva y MineClawd te avisará cuando pidas una búsqueda web.

## Paso 6: envía tu primer mensaje

Tienes dos formas de hablar con MineClawd:

### Opción A: usa la GUI de superposición

Busca el orbe flotante (Assistive Touch) en el lado izquierdo de la pantalla. Haz clic en el orbe para abrir la superposición, escribe tu mensaje en la barra de entrada y haz clic en **Send**.

<Frame caption="El orbe Assistive Touch abre la superposición con un clic">
  <img src="/images/mineclawd-assistive-touch.png" alt="Orbe flotante Assistive Touch de MineClawd" />
</Frame>

### Opción B: usa comandos de chat

Abre el chat y ejecuta:

```bash
/mclawd hello
```

Ambos métodos funcionan. La superposición te da respuestas en streaming y gestión de Session en un solo lugar. Revisa [uso de la GUI](/tutorials/using-the-gui) para la guía completa.
