## Reglas globales

- Todos los comandos de MineClawd requieren nivel de permiso OP 2.
- El comando raíz es `/mineclawd`.
- El alias rápido para prompt es `/mclawd <request>`.
- Las referencias de sesión aceptan id de sesión (`1a2b`) o token (`1a2b-new-session`).

## Mapa de comandos

| Comando | Propósito | Referencia |
| --- | --- | --- |
| `/mineclawd prompt <request>` | Envía una solicitud a MineClawd. | [/commands/prompt](/commands/prompt) |
| `/mclawd <request>` | Alias corto para prompt. | [/commands/prompt](/commands/prompt) |
| `/mineclawd config ...` | Abre, consulta o actualiza la configuración. | [/commands/config](/commands/config) |
| `/mineclawd sessions new` | Crea y cambia a una sesión nueva. | [/commands/sessions-new](/commands/sessions-new) |
| `/mineclawd sessions list` | Muestra todas las sesiones guardadas. | [/commands/sessions-list](/commands/sessions-list) |
| `/mineclawd sessions resume <session>` | Cambia la sesión activa. | [/commands/sessions-resume](/commands/sessions-resume) |
| `/mineclawd sessions remove <session>` | Elimina una sesión. | [/commands/sessions-remove](/commands/sessions-remove) |
| `/mineclawd sessions repair [session]` | Repara turnos de historial de Vertex mal formados. | [/commands/sessions-repair](/commands/sessions-repair) |
| `/mineclawd history` | Abre la interfaz en libro del historial de la sesión actual. | [/commands/history](/commands/history) |
| `/mineclawd retry <token>` | Reintenta la última solicitud fallida. | [/commands/retry](/commands/retry) |
| `/mineclawd persona [soul]` | Muestra o cambia la personalidad soul activa. | [/commands/persona](/commands/persona) |
| `/mineclawd choose [option]` | Responde preguntas de aclaración pendientes. | [/commands/choose](/commands/choose) |
| `/mineclawd new` | Alias de `sessions new`. | [/commands/new](/commands/new) |
| `/mineclawd stop` | Cancela la generación activa. | [/commands/stop](/commands/stop) |
| `/mineclawd assets ...` | Lista, teletransporta, da o elimina activos rastreados. | [/commands/assets](/commands/assets) |
| `/mineclawd assistivetouch` | Alterna la visibilidad del orbe flotante. | [/commands/assistivetouch](/commands/assistivetouch) |
