---

## 目的

現在実行中のリクエストを取り消します。

## 構文

```bash
/mineclawd stop
```

## 使うタイミング

- リクエストが長すぎると感じたとき。
- MineClawd がツール呼び出しループに入り込んだとき。
- 生成中にリクエスト内容を変更したくなったとき。

## 動作

1. アクティブなリクエストをキャンセル済みとしてマークします。
2. LLM プロバイダーへの進行中のネットワーク通信を停止します。
3. 保留中の `ask-user-question` プロンプトを破棄します。
4. セッションのロックを解除し、新しいリクエストを送れるようにします。

## GUI での代替

リクエスト実行中はオーバーレイの送信ボタンが **Stop** ボタン（四角アイコン付き）に変わります。クリックすると `/mineclawd stop` と同じ動作です。

## 動作制約

- 実行中のリクエストがない場合は何も起きません。
- キャンセル前にストリーミングされた部分応答は履歴に残ります。
- セッションはそのまま維持されます。

## よくある失敗

- `No active request to cancel` -- アクティブな生成が存在しません。
