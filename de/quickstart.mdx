## Ziel

Folge dieser Seite Schritt für Schritt, und du bekommst eine erfolgreiche Antwort von MineClawd.

## Schritt 3: Mod-Dateien in deinen `mods`-Ordner legen

1. Öffne deinen Minecraft-Ordner.
2. Öffne den `mods`-Ordner. Wenn er nicht existiert, erstellst du ihn.
3. Verschiebe alle heruntergeladenen `.jar`-Dateien nach `mods`.
4. Entpacke die `.jar`-Dateien nicht.

## Schritt 4: Spiel starten und MineClawd-Konfiguration öffnen

1. Starte Minecraft mit deinem Loader-Profil.
2. Erstelle oder öffne eine Einzelspielerwelt.
3. Schalte **Allow Cheats** für diese Welt auf ON.
4. Öffne **Better ModList** -> **MineClawd** -> **Configure** (oder **Mod Menu**, wenn du Fabric nutzt).

## Schritt 5: AI-Service verbinden

Wenn du noch kein AI-Konto hast, nutze OpenRouter und wähle ein kostenloses Modell.

1. Konto erstellen: [openrouter.ai](https://openrouter.ai/)
2. API-Key erstellen: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)
3. In der MineClawd-Konfiguration setzen:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: Deinen OpenRouter-Key einfügen
   - **Model**: Eine Modell-ID von [openrouter.ai/models](https://openrouter.ai/models) auswählen (kostenlose Modelle enthalten oft `:free`)
   - **Summarize Model**: Vorerst dieselbe Modell-ID verwenden
4. Konfiguration speichern.

<Info>
Kostenlose Modelle und Limits ändern sich mit der Zeit. Wenn ein Modell fehlschlägt, probiere eine andere Modell-ID von derselben Seite.
</Info>

## Schritt 6: Deine erste Nachricht senden

Du kannst auf zwei Wegen mit MineClawd sprechen:

### Option A: Overlay-GUI verwenden

Suche den schwebenden Orb (AssistiveTouch) links auf dem Bildschirm. Klicke auf den Orb, um das Overlay zu öffnen, gib deine Nachricht in die Eingabeleiste ein und klicke auf **Send**.

<Frame caption="Der AssistiveTouch-Orb öffnet das Overlay mit einem Klick">
  <img src="/images/mineclawd-assistive-touch.png" alt="MineClawd AssistiveTouch-Orb auf dem Bildschirm" />
</Frame>

### Option B: Chatbefehle verwenden

Öffne den Chat und führe aus:

```bash
/mclawd hello
```

Beide Wege funktionieren. Das Overlay zeigt Streaming-Antworten und Session-Management in einem Fenster. Sieh dir [die GUI verwenden](/tutorials/using-the-gui) an, um die komplette Anleitung zu bekommen.
