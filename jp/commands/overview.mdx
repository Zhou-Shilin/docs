## グローバルルール

- すべての MineClawd コマンドには OP 権限レベル 2 が必要です。
- コマンドルートは `/mineclawd` です。
- prompt の短縮エイリアスは `/mclawd <request>` です。
- セッション参照はセッション ID（`1a2b`）または token（`1a2b-new-session`）のどちらも使えます。

## コマンドマップ

| コマンド | 目的 | リファレンス |
| --- | --- | --- |
| `/mineclawd prompt <request>` | MineClawd にリクエストを送信します。 | [/commands/prompt](/commands/prompt) |
| `/mclawd <request>` | prompt の短縮エイリアスです。 | [/commands/prompt](/commands/prompt) |
| `/mineclawd config ...` | 設定を開く・確認する・更新する。 | [/commands/config](/commands/config) |
| `/mineclawd sessions new` | 新しいセッションを作成して切り替える。 | [/commands/sessions-new](/commands/sessions-new) |
| `/mineclawd sessions list` | 保存済みセッションを一覧表示する。 | [/commands/sessions-list](/commands/sessions-list) |
| `/mineclawd sessions resume <session>` | アクティブセッションを切り替える。 | [/commands/sessions-resume](/commands/sessions-resume) |
| `/mineclawd sessions remove <session>` | セッションを削除する。 | [/commands/sessions-remove](/commands/sessions-remove) |
| `/mineclawd sessions repair [session]` | 破損した Vertex 履歴ターンを修復する。 | [/commands/sessions-repair](/commands/sessions-repair) |
| `/mineclawd history` | 現在のセッション履歴を本 UI で開く。 | [/commands/history](/commands/history) |
| `/mineclawd retry <token>` | 直近の失敗リクエストを再試行する。 | [/commands/retry](/commands/retry) |
| `/mineclawd persona [soul]` | アクティブ soul を表示または切り替える。 | [/commands/persona](/commands/persona) |
| `/mineclawd choose [option]` | 保留中の確認質問に回答する。 | [/commands/choose](/commands/choose) |
| `/mineclawd new` | `sessions new` のエイリアス。 | [/commands/new](/commands/new) |
