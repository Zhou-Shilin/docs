---
title: "sessions list"
description: "列出当前 owner 已保存的 MineClawd 会话。"
---

## 用途

显示你保存的会话，便于你恢复、删除或修复。

## 语法

```bash
/mineclawd sessions
/mineclawd sessions list
```

## 返回内容

列表包含：

- 会话 id
- 会话标题
- 会话 token
- 最后更新时间
- 活动标记

## 示例

```bash
/mineclawd sessions list
```

## 行为和限制

- 会话按 owner key 隔离。
- 玩家执行时使用玩家 UUID 作为 owner key。
- 控制台执行时使用 `server` 作为 owner key。

## 常见失败

- 如果没有任何会话，请先运行 `/mineclawd sessions new` 创建一个。
