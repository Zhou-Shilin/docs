---
title: "history"
description: "Eine lesbare In-Game-Buchansicht des aktuellen Session-Verlaufs öffnen."
---

## Zweck

Zeige die Konversation deiner aktiven Session in einer schriftlichen Buch-UI an.

## Syntax

```bash
/mineclawd history
```

## Wann du es verwendest

Nutze das, um den Kontext zu prüfen, bevor du eine lange Aufgabe fortsetzt.

## Verhalten und Grenzen

- Dieser Befehl muss von einem Spieler ausgeführt werden, nicht von der Konsole.
- Für das direkte Verlauf-Buch-UI-Paket ist Client-Unterstützung erforderlich.
- Wenn es keine aktive Session gibt, fordert MineClawd dich auf, zuerst eine zu erstellen.
- Sehr lange Verläufe werden auf die Buchlimits gekürzt.

## Häufige Fehler

- `this command must be executed by a player`: im Spieler-Chat ausführen.
- `Client mod is required for /mineclawd history book UI`: MineClawd auf deinem Client installieren.
- `No active session`: zuerst `/mineclawd sessions new` ausführen.
