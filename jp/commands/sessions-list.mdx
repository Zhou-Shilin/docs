---
title: "sessions list"
description: "現在の owner に保存された MineClawd セッションを一覧表示します。"
---

## 目的

保存済みセッションを表示し、resume、remove、repair を行えるようにします。

## 構文

```bash
/mineclawd sessions
/mineclawd sessions list
```

## 表示内容

一覧には次が含まれます:

- セッション ID
- セッションタイトル
- セッショントークン
- 最終更新時刻
- アクティブマーカー

## 例

```bash
/mineclawd sessions list
```

## 動作と制限

- セッションは owner key ごとに分離されます。
- プレイヤー実行ではプレイヤー UUID が owner key になります。
- コンソール実行では `server` が owner key になります。

## よくある失敗

- セッションが表示されない場合は、`/mineclawd sessions new` で作成してください。
