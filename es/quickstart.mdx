## Objetivo

Sigue esta página paso a paso y obtendrás una respuesta correcta de MineClawd.

## Paso 1: instala Fabric para Minecraft 1.20.1

1. Descarga Fabric Installer desde: [fabricmc.net/use/installer](https://fabricmc.net/use/installer/)
2. Ejecuta el instalador.
3. Elige:
   - Tipo de instalación: **Client**
   - Versión de Minecraft: **1.20.1**
4. Haz clic en **Install**.
5. Abre Minecraft Launcher y ejecuta **Fabric Loader 1.20.1** una vez, luego cierra el juego.

<Note>
En Windows con el launcher oficial, esto suele ser un flujo de instalación de un solo clic.
</Note>

## Paso 2: descarga los mods requeridos (versiones Fabric + 1.20.1)

Descarga estos archivos desde Modrinth. En cada página, elige un archivo que coincida con:

- Versión de Minecraft: `1.20.1`
- Loader: `Fabric`

| Mod | Enlace de descarga |
| --- | --- |
| MineClawd | [modrinth.com/mod/mineclawd](https://modrinth.com/mod/mineclawd) |
| Fabric API | [modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api) |
| KubeJS | [modrinth.com/mod/kubejs](https://modrinth.com/mod/kubejs) |
| Architectury API | [modrinth.com/mod/architectury-api](https://modrinth.com/mod/architectury-api) |
| Rhino | [modrinth.com/mod/rhino](https://modrinth.com/mod/rhino) |
| YetAnotherConfigLib v3 (YACL) | [modrinth.com/mod/yacl](https://modrinth.com/mod/yacl) |
| Mod Menu (para configuración GUI) | [modrinth.com/mod/modmenu](https://modrinth.com/mod/modmenu) |

## Paso 3: coloca los archivos del mod en tu carpeta `mods`

1. Abre tu carpeta de Minecraft.
2. Abre la carpeta `mods`. Si no existe, créala.
3. Mueve todos los archivos `.jar` descargados a `mods`.
4. No descomprimas los archivos `.jar`.

## Paso 4: inicia el juego y abre la GUI de MineClawd

1. Inicia Minecraft con el perfil **Fabric Loader 1.20.1**.
2. Crea o abre un mundo de un solo jugador.
3. Activa **Allow Cheats** en ese mundo.
4. Abre **Mod Menu** -> **MineClawd** -> **Configure**.

## Paso 5: conecta un servicio de IA

Si todavía no tienes una cuenta de IA, usa OpenRouter y elige un modelo gratuito.

1. Crea una cuenta: [openrouter.ai](https://openrouter.ai/)
2. Crea una clave API: [openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)
3. En la GUI de MineClawd, configura:
   - **Provider**: `OpenAI`
   - **Endpoint**: `https://openrouter.ai/api/v1`
   - **API Key**: pega tu clave de OpenRouter
   - **Model**: elige un id de modelo de [openrouter.ai/models](https://openrouter.ai/models) (los modelos gratis suelen incluir `:free`)
   - **Summarize Model**: usa por ahora el mismo id de modelo
4. Guarda la configuración.

<Info>
Los modelos gratuitos y sus límites cambian con el tiempo. Si falla un modelo, elige otro id de modelo de la misma página.
</Info>

## Paso 6: envía tu primer mensaje

Abre el chat en tu mundo y ejecuta:

```bash
/mclawd hola
```

Si la configuración es correcta, MineClawd responderá en el chat.
